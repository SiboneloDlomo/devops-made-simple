# devops-made-simple
# A beginner-friendly guide demonstrating how to automate CI/CD pipelines using GitHub Actions and deploy applications to Kubernetes.

This project walks you through creating a basic CI/CD pipeline using GitHub Actions, Docker, and Kubernetes. It's designed for beginners in DevOps.

### Key Concepts Covered:
- Continuous Integration (CI)
- Continuous Deployment (CD)
- GitHub Actions workflows
- Docker image builds
- Kubernetes deployments

---

## Setup Overview:
1. Write and push your code
2. GitHub Actions builds and pushes Docker images
3. Deployment happens automatically to Kubernetes

See the full guide in the [documentation](./docs/guide.md).
